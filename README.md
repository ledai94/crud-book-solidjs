### App

A Basic SolidJS App with CRUD operations

### Tech Stack

- **Client:** SolidJS
- **Server:** Node
- **Styling**: Styled Components
- **Router**: Solid App Router
- **Store data**: Local Storage

### Concepts covered

1. Local State
2. Context API (with createContext/Context Provider)
3. Hooks (createSignal,createEffect)
